# Simple schedule system for job test 

### How to use/import

- First, clone it
```bash
git clone https://github.com/DarknessNinjaBR/tecimob-test-schedule.git
```
- install dependencies
```bash
composer install
```
Copy .env.example file to .env on the root folder and edit has indeed.
After, Run 
```bash
php artisan migrate
```
and finally run
```bash
php artisan serve
```
